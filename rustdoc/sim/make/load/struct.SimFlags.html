Expand description

SimFlags specifies a simulation to setup. After parsing from structopt, you must call initialize.

Fieldsยง

ยงload_path: Option<String>

A path to some file:

  • some kind of map: start an empty simulation on the map
  • a scenario
  • a savestate: restore the simulation exactly from some savestate
ยงload: String

The same as load_path, but with a default value filled out. Call initialize to set this up.

ยงscenario_modifiers: Vec<ScenarioModifier>

A JSON list of modifiers to transform the scenario. These can be generated with the GUI.

ยงrng_seed: u64

An arbitrary number to seed the random number generator. This is input to the deterministic simulation, so different values affect results.

ยงopts: SimOptions
