pub fn cli_args() -> impl Iterator<Item = OsString>
Expand description

Returns arguments passed in from the command-line, starting with the program name.

On web, this transforms URL query parameters into command-line arguments, by treating & as the separator between arguments. So for instance “?–dev&–color_scheme=night%20mode” becomes vec![“dummy program name”, “–dev”, “–color_scheme=night mode”].
